## ğŸ¨ Customization

### Change Colors

Edit the CSS variables in the `<style>` section:

```css
:root {
    --bg-primary: #0a0e1a;
    --accent-green: #00ffa3;
    --accent-cyan: #00d4ff;
    /* Customize these! */
}
```

### Change Fonts

Replace the Google Fonts link in the HTML `<head>`:

```html
<link href="https://fonts.googleapis.com/css2?family=YOUR_FONT&display=swap" rel="stylesheet">
```

### Modify AI Prompt

Edit `YOUR_PROMPT` in `generate_company_data.py` to customize what information is generated.

## â“ Troubleshooting

**Can't find Python?**
- Try `python3` instead of `python`
- Install from python.org

**API key not working?**
- Make sure it's set: `echo $XAI_API_KEY`
- No quotes in the actual key

**Company not found on website?**
- Check `data/` folder exists
- Make sure ticker.json file is there
- Verify filename is uppercase
